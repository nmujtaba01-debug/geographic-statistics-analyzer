# Geographic Statistics Analyzer

A Python project that analyzes geographic datasets using statistical calculations such as Average (AVG), Maximum (MAX), and Minimum (MIN).

## Features

- Calculate Average Population
- Calculate Average Rainfall
- Find Highest Population
- Find Lowest Population
- Find Highest Rainfall
- Find Lowest Rainfall
- Perform multiple analyses in a single loop

## Dataset

The project uses a geographic dataset containing:

- City Names
- Population Data
- Rainfall Data

Example:

```python
places = [
    {'name':'Delhi', 'population':19000000, 'rainfall':80},
    {'name':'Mumbai', 'population':20000000, 'rainfall':220},
    {'name':'Jaipur', 'population':4000000, 'rainfall':60},
    {'name':'Shillong', 'population':500000, 'rainfall':250},
    {'name':'Chennai', 'population':11000000, 'rainfall':140}
]
```

## Concepts Used

- Python Lists
- Dictionaries
- For Loops
- Conditional Statements
- Average Calculation (AVG)
- Maximum Value Analysis (MAX)
- Minimum Value Analysis (MIN)
- Data Aggregation
